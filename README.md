# 100 Days of MLOps

A hands-on MLOps engineering challenge focused on building production-grade machine learning workflows and infrastructure.

---

## Skills Covered

- Python
- MLflow
- DVC
- Docker
- Kubernetes
- FastAPI
- CI/CD
- Monitoring
- Model Serving
- Feature Stores
- Experiment Tracking
- Infrastructure Automation

---

## Goals

- Build strong MLOps fundamentals
- Learn production ML workflows
- Practice real-world DevOps + ML integration
- Create an industry-ready portfolio
